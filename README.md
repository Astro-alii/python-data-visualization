# Python Data Visualization

A hands-on learning repo following the *Data Visualization* chapters of **Python Crash Course** by Eric Matthes. Covers matplotlib, Plotly, and working with real-world downloaded datasets.

---

## What's Covered

### Matplotlib
- Line charts and scatter plots
- Styling plots — colors, markers, labels, titles
- Subplots and figure sizing
- Numpy integration — `linspace`, vectorized operations, trig functions
- Built-in matplotlib styles

### Plotly
- Interactive bar and line charts
- Hover data and layout customization
- Comparing static (matplotlib) vs interactive (Plotly) visualization

### Downloading & Visualizing Real Data
- Parsing CSV and JSON datasets
- Plotting weather data, earthquake data, and more
- Working with `csv`, `json`, and `datetime` modules
- Building charts from real-world sources (NOAA, USGS, etc.)
